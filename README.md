# 🎲 Lunch Picker (Random Name Picker)

A simple web app that randomly selects a name from a list.
It is useful for making quick random decisions such as picking a person, choosing a winner, or selecting someone from a group.

🔗 **Live Demo:**
https://bahadirbektore.github.io/lunch-picker/

---

## 📌 Project Description

Random Name Picker is a small JavaScript project that randomly selects one name from a list entered by the user.

The application demonstrates how to work with **JavaScript arrays, random number generation, and DOM manipulation** to create an interactive user experience.

---

## 🚀 Features

* Randomly select one name
* Simple and clean user interface
* Instant result generation
* Beginner-friendly JavaScript logic

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**

The app uses JavaScript's random number generation to select a name from an array created from the user's input.

---

## 🎯 Purpose of the Project

This project was built to practice:

* DOM manipulation
* Handling user input
* JavaScript arrays
* Random selection logic
* Creating small interactive web apps
